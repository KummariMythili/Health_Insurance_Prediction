About the Project

This machine learning model helps predict whether a submitted health insurance claim is likely to be genuine or fraudulent based on key input features.

📊 Feature Inputs

Analyzes age, gender, BMI, smoker status, claim amount, etc.

🧠 ML Algorithms

Built using algorithms like Logistic Regression, SVM, and Random Forest.

🎯 Claim Decision

Predicts if the claim is genuine or potentially fraudulent.

📘 Understanding the Fields

  • Age: Age of the policyholder.
  • Gender: 1 = Male, 0 = Female.
  • BMI: Body Mass Index (health factor).
  • Smoker: 1 = Smoker, 0 = Non-smoker.
  • Dependents: Number of dependents on the policy.
  • Claim Type: Encoded claim type.
  • Claim Amount: Amount claimed by the insured.
  • Previous Claims: Number of previous claims filed.
  • Stay Duration: Duration of hospital stay.
  • Doctor Visits: Number of doctor visits for treatment.
